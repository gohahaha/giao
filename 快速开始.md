## 什么是新版？

现在每个人都有自己的账号，发的动态、照片、留言都**存在云端**，所有人的数据**实时共享**！

---

## 🚀 第一步：注册 Supabase（免费云端数据库）

1. 打开 [app.supabase.com](https://app.supabase.com)
2. 点击 **Sign in with GitHub**（用 GitHub 账号登录，没有的话先注册一个 GitHub）
3. 点击 **New project** 创建新项目
4. 填写：
   - Name: `giaogiao-house`
   - Database Password: 设置一个密码（记住它）
   - Region: 选 **Singapore**（离国内最近，访问最快）
   - 点击 **Create project**
5. 等待 1-2 分钟创建完成

## 📋 第二步：创建数据库表

1. 在 Supabase 控制台左侧菜单，点击 **SQL Editor**
2. 点击 **New query**
3. 打开项目里的 `supabase-setup.sql` 文件
4. 复制全部内容，粘贴到 SQL Editor
5. 点击右下角 **Run** 执行
6. 看到 "Success" 就 OK 了

## 🔑 第三步：配置 API 密钥

1. 在 Supabase 控制台左侧，点击 **Settings** → **API**
2. 复制这两个值：
   - **Project URL**（类似 `https://xxxxx.supabase.co`）
   - **anon public key**（很长一串）
3. 打开项目里的 `js/supabase-config.js` 文件
4. 把两个值粘贴进去：
   ```javascript
   const SUPABASE_URL = 'https://xxxxx.supabase.co';  // 你的 Project URL
   const SUPABASE_ANON_KEY = '你的 anon key';         // 你的 anon key
   ```

## 📦 第四步：部署到网上

### 方法一：GitHub Pages（推荐，免费）

1. 创建 GitHub 账号 → 新建仓库叫 `house`
2. 把所有文件上传到仓库
3. Settings → Pages → Source 选 `main` 分支 → Save
4. 1-2 分钟后访问 `https://你的用户名.github.io/house/`

### 方法二：直接发给朋友

把所有文件打包成 zip，发给朋友们，他们解压后用浏览器打开 `index.html` 即可。

## 👥 第五步：注册账号

1. 打开网站，看到登录页面
2. 点击 **注册加入**
3. 输入邮箱 + 密码 + 选择你是哪位成员
4. 注册成功后自动登录
5. 10 个人各自注册一次即可
