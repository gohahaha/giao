## 📝 修改成员信息

打开 `js/data.js` 文件，找到 `MEMBERS` 数组，修改每个人的信息：

```javascript
const MEMBERS = [
    {
        id: 1,                    // 唯一ID，不要重复
        name: '米奇',             // 昵称
        emoji: '🐭',              // 头像表情（可以换成其他emoji）
        title: '群主·小屋创始人',  // 头衔
        desc: '最爱你们的米奇',    // 个人简介
        joinDate: '2024-01-01',   // 入坑时间
        motto: '有你们在，每天都是好日子' // 座右铭
    },
    // ... 继续添加其他成员
];
```

## 🎨 修改颜色主题

打开 `css/style.css` 文件，找到 `:root` 部分：

```css
:root {
    /* 主色调 */
    --primary: #F5E6D3;        /* 奶杏色 */
    --primary-light: #FFF8F0;  /* 浅奶白 */
    --primary-dark: #E8D5C0;   /* 深奶杏 */

    /* 点缀色 */
    --accent: #C41E3A;         /* 复古红 */
    --accent-light: #FF6B6B;   /* 浅红 */

    /* 中性色 */
    --text: #2C2C2C;           /* 浅黑 */
    --text-light: #666666;
    --text-muted: #999999;
    --bg: #FFFAF5;             /* 米白背景 */
}
```


## ❓ 常见问题

### Q: 如何备份数据？
A: 打开浏览器开发者工具（F12），在 Console 中输入：
```javascript
copy(JSON.stringify(localStorage))
```

### Q: 如何恢复数据？
A: 在 Console 中输入：
```javascript
localStorage.setItem('house_feed', '你的数据')
```

### Q: 数据会丢失吗？
A: 数据存在浏览器中，以下情况会丢失：
- 清除浏览器数据
- 换浏览器访问
- 换电脑访问

建议定期备份重要数据！

### Q: 如何让所有人看到相同的数据？
A: 目前数据存在各自的浏览器中。如需共享数据，需要：
1. 使用后端服务器（需要编程知识）
2. 或使用云存储服务
